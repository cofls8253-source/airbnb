# 서울 에어비앤비 블루오션 발굴 전략 대시보드 (v9.0)

서울 에어비앤비 창업을 고민하는 예비 창업자를 위한 데이터 기반 의사결정 지원 대시보드입니다. 
리포트의 인사이트(1,263건 유효 데이터)를 바탕으로 수익성 높은 입지를 제안합니다.

## 🚀 주요 기능
- **통합 대시보드 메트릭**: 평균가, 숙소 수 등 핵심 지표 요약.
- **레드/블루오션 지도 (Plotly)**: 공급 과열 지구와 기회 지구 시각화.
- **수급 불균형 분석**: 구별 숙소 수 vs 평균 가격 비교.
- **투자 회수 시뮬레이터**: 투자금 대비 회수 기간 자동 계산.
- **오늘의 블루오션 추천**: 알고리즘 기반 최적 입지 TOP 3 제안.

## 🛠 설치 및 실행 방법
1. 저장소를 클론합니다:
   ```bash
   git clone https://github.com/your-username/seoul-airbnb-dash.git
   ```
2. 필수 라이브러리를 설치합니다:
   ```bash
   pip install -r requirements.txt
   ```
3. 스트림릿을 실행합니다:
   ```bash
   streamlit run app.py
   ```

## 📊 데이터 소스
- `airbnb.db`: 서울 에어비앤비 숙소 데이터베이스 (SQLite3)
- 분석 대상: 1,263건의 정밀 샘플링 데이터
